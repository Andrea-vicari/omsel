OMSEL Onepage React Portfolio

Omsel is a clean, modern React Portfolio website. It can be used to showcase your projects or your service. It is built with Bootstrap5 to ensure compatibility accross devices. It has a filterable portfolio section, easily customizable editing a local Json file. It also has a working contact form, to receive email without a Database. It can be used as a landing page, as a portfolio or resume website.

FEATURES

  • React JS
  • Bootstrap5
  • Ligth-Dark mode
  • Sass files for easy customization
  • Filterable Json portfolio
  • Setup & Live Deployment video-instructions
  • Working EmailJS Contact Form
  • React Functional components
